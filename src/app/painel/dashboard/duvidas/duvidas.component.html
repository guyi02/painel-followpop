1. Tire suas dúvidas

- Como fazer um novo pedido?

na aba lateral(desktop) ou superior (mobile) existe os serviços para cada rede social específica.

no campo Link - preencha o link para a página no formato indicado no exemplo.

no campo Quantidade - preencha a quantidade necessária de curtidas / seguidores respeitando o formato indicado.



- Instruções

*Aviso: Ao fazer um pedido, não coloque o mesmo link ao mesmo tempo nos mesmos serviços ou serviços diferentes até que o pedido seja totalmente concluído ou cancelado. Exemplo se você fizer um pedido para o mesmo perfil ao mesmo tempo, ele contará apenas UM PEDIDO, não será possível obter seguidores para os dois pedidos, então aguarde um tempo antes de enviar. Não compre seguidores / curtidas de outros sites, isso pode causar conflitos.
